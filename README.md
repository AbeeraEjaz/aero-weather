## ✨ Features

### 🌤️ Free Features (No Login Required)

| Feature | Description |
|:---|:---|
| ✅ **Real-Time Weather** | Live weather data for any city worldwide |
| ✅ **7-Day Forecast** | Daily weather predictions for the week |
| ✅ **Hourly Forecast** | Next 24 hours weather breakdown |
| ✅ **Dynamic Backgrounds** | Weather-based beautiful gradients |
| ✅ **Unit Toggle** | Switch between Celsius and Fahrenheit |
| ✅ **Location Detection** | Auto-detect your current city using GPS |

### ⭐ Premium Features (Login Required)

| Feature | Description |
|:---|:---|
| ❤️ **Favorites** | Save unlimited favorite cities |
| 💧 **Skin & Hair Care Tips** | Weather-based beauty and skincare advice |
| 🎯 **Weather Score** | AI-powered comfort rating (0-100) |
| 🌅 **Best Time to Go Out** | Optimal outdoor timing based on weather |
| ⚡ **Weather Fun Facts** | Interesting daily weather facts |
| 📊 **Recent Searches** | History of your city searches |
| 🔒 **Cloud Sync** | Your data synced across all devices |

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- OpenWeatherMap API Key
- Firebase Account (for authentication)

### Installation

```bash
# Clone the repository
git clone https://github.com/AbeeraEjaz/aero-weather.git
cd aero-weather

# Install dependencies
npm install

# Create .env file with your keys
# Copy from .env.example and add your keys

# Start development server
npm run dev
